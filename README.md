# Data Flow >> 

##### A powerful, user-friendly web interface for managing MySQL databases without writing SQL queries. DBFlow provides complete CRUD operations through an intuitive Streamlit-based GUI.


## ✨ Features

* 🎯 **Dynamic Table Creation** - Design custom tables with various data types
* 📊 **Data Visualization** - Built-in charts and statistical analysis
* 🔄 **Complete CRUD Operations** - Create, Read, Update, Delete records
* 📤 **Export Functionality** - Download data as CSV files
* 🔍 **Schema Introspection** - View table structures and metadata
* 🛡️ **Type Safety** - Automatic data validation and type conversion
* 🎨 **Responsive UI** - Modern, intuitive web interface


## Configuration

> Database Settings:  Change the config according to your DB


```
host='localhost'
port=3306
user='root'
password='your_password'
database='techstack'
```

> Run using 

```
streamlit run app.py
```
